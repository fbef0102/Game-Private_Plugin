# Description | 內容

# 中文說明
輸入!trade打開選單選擇雙方隊伍一位玩家，然後全體投票決定兩位玩家交換隊伍

* 原理
	* 輸入!trade打開選單選擇己方一位玩家，接著選擇對方一位玩家，選完之後全體玩家投票，投票通過後，兩位玩家互換陣營
	* 離開安全室之後就不能投票了
	* 覺得隊友太爛，對面隊伍太強，實力不平衡可以使用這插件

* 功能
	* 設置投票間隔時間

# 中文說明
傷害比例調整插件，可自行調整 Tank/Witch/特感/小殭屍/跳樓 對人類造成的傷害比

* 功能
	* 可調整人類所有傷害比，也可以設置成完全不受到傷害
	* 可調整倒地受傷害比以及掛邊的受傷害比

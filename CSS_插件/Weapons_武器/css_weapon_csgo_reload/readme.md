# 中文說明
將CS武器改成現代遊戲的裝子彈機制 (仿CS2切槍裝彈設定)

* 適用於
	```
	絕對武力：次世代
	```

* 原理
    * 當武器動畫是裝上彈夾的時候，彈夾會填滿 (就像CSGO那樣)    
    * 裝彈切槍，耍酷用

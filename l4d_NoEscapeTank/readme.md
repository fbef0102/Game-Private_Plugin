# 中文說明
救援載具來臨之後不會有Tank來襲

* 原理
	* 當最後直升機、船隻、火車等救援來臨之後，不會有Tank來襲
	* 也適用於對抗，真人玩家不會有Tank能操控

* 功能
	1. 設置插件開關
